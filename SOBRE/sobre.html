Sobre os Desenvolvedores e o Projeto CryptoInfo

Esta plataforma digital foi concebida e desenvolvida por um grupo de estudantes da Fatec de São Bernardo do Campo (SP), como parte de um projeto acadêmico focado em disseminar conhecimento especializado.

Objetivo da Plataforma

O principal objetivo do projeto CryptoInfo é atuar como uma fonte de informação clara e acessível sobre Criptografia e sua relevância fundamental para a Cibersegurança moderna. Acreditamos que o entendimento desses mecanismos é essencial tanto para estudantes da área de tecnologia quanto para qualquer indivíduo interessado em compreender a segurança digital no cotidiano. O site visa desmistificar algoritmos complexos e ilustrar sua aplicação prática no mundo real.

Tecnologias Utilizadas no Desenvolvimento

O CryptoInfo foi construído integralmente utilizando o ambiente de desenvolvimento Visual Studio Code (VS Code), empregando uma pilha de tecnologias focada na leveza, acessibilidade e robustez da interface.

As principais linguagens e bibliotecas que compõem o código-fonte incluem:

  • HTML5: Estrutura fundamental de todo o conteúdo da plataforma.
  • CSS3 (Tailwind CSS): Estilização e responsividade da interface. O framework Tailwind CSS foi essencial para a aplicação de um design moderno e consistente.
  • JavaScript (JS): Linguagem responsável por toda a interatividade e a lógica de simulação didática.

Para funcionalidades avançadas e gráficos, foram integradas as seguintes bibliotecas:

  • Three.js: Utilizada para a criação de elementos 3D e experiências visuais imersivas (como o plano de fundo do canvas).
  • Crypto-js: Implementada nas seções de simulação, fornecendo uma base para as operações de hashing e manipulação de cifras simétricas (como DES e AES) de forma didática.

O projeto demonstra a aplicação prática de conceitos de Desenvolvimento Web na criação de ferramentas educativas de alto valor.
